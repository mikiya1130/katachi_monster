# カタチモンスター(Shape Monster)

- カタチを見つけて自分だけのモンスターを作成
- 一致率が高いほど、攻撃力が向上
- 勝った際の攻撃力を優先させるか、負けた際のダメージを低減させるか、戦略的なじゃんけんバトル
- 子供の活動性・創造性を高めます

CEDC(Creative Engineering Design Competition)2023 最優秀賞 受賞
